NETGEAR R6100 安装lEDE

​ 最近准备学习一下LEDE，度娘比较推荐网件WNDR4300 v1，但这货已经停产，能买到的只有v2 。只有折腾 R6100了。

原厂系统

​ 用了几天原厂系统，网速还比较满意，管理端响应非常慢。

方法

​ 官方提供四种安装方法Installing OpenWrt 。我使用Method 1: via OEM firmware安装LEDE，选择Method 2: via Bootloader and an Ethernet port 恢复原厂固件。

安装

  • 下载 R6100 固件lede-17.01.4-ar71xx-nand-r6100-ubi-factory.img

  • 网线连接路由器LAN口

  • 打开路由器管理端

  • 点击管理-固件升级

  • 选择步骤1下载的固件，上传

  • 点击安装，waiting…

  • 重新获取到IP后，登陆管理端

  • 第一次登陆没有密码

  • 登陆后可以看到状态信息

恢复原厂固件

​ 想要恢复原厂固件也是可以的，步骤如下

  • 网线连接路由器LAN口

  • 网件中文官网下载固件R6100-V1.2.0.4.img，并放在用户目录(eg. C:\Users\apple)下

  • 开启TFTP client

  • 打开cmd，输入ping 192.168.1.1 –t

  • 再打开一个cmd，输入TFTP -i 192.168.1.1 PUT R6100-V1.2.0.4.img，不要回车

  • 戳reset不放，拔掉电源再接上，看第一个cmd窗口ping通了后，在第二个cmd窗口回车

  • 第二个cmd窗口出现传输成功提示后，松开reset

  • waiting…

  • 重新获取到IP后，打开管理端确认

有什么用呢

敲黑板

  • LEDE是openwrt的一个分支

  • WNDR4300 v1可以安装openwrt/lEDE，但已经停产

  • WNDR4300 v2 目前不能安装openwrt/lEDE

  • R6100 可以安装 openwrt/lEDE

  • R6100 5G信号问题已经解决
