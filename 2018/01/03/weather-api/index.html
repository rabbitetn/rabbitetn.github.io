使用OpenWeatherMap API获取天气数据

发表于 2018-01-03 | 分类于 Tech

​ OpenWeatherMap是美帝的一家IT公司，主要从事大数据、地理空间技术。其向开发者免费提供天气API，可以查询历史、实时天气数据，并提供天气预测等功能。API支持JSON, XML, or HTML 格式，具有较高的可用性。

注册

​ 使用API之前，需要先注册OpenWeatherMap账号。点击Create an Account.输入简单的信息就可完成注册了。

​

API key

​ 登陆OpenWeatherMap，复制API KEY，以备使用。

API call

​ 根据API文档生成API调用地址(by city ID）

http://api.openweathermap.org/data/2.5/weather?id=1796236&units=metric&appid=***

  • id为城市id
  • units为温度单位，metric表示摄氏度
  • appid即API KEY

​

​ 打开Postman，输入API调用地址，即可查询到天气情况

微服务

​ 将weather api整合到微服务中,主要类如下

  • LabApplication.java
  • WeatherController.java
  • WeatherService.java
  • OpenWeatherMap.java
